# yzkongjian

Astro 静态站点项目。

## 开发命令

| Command           | Action                                      |
| :---------------- | :------------------------------------------ |
| `npm install`     | 安装依赖                                    |
| `npm run dev`     | 本地开发，默认 `localhost:4321`             |
| `npm run build`   | 构建生产版本到 `./dist/`                    |
| `npm run preview` | 本地预览构建结果                            |
